## 📝 简介

我已经测试了许多开源 LLM 模型生成 Cypher 语句的能力。其中的一些对 Cypher 语法有基本的了解。但是，我没有找到任何模型能够根据提供的示例或图形架构可靠地生成 Cypher 语句。因此，唯一的解决方案是微调开源 LLM 模型以可靠地生成 Cypher 语句。故本项目使用书生大模型完整的体系来构建一个知识图谱问答系统。
